# 🌤️ Weather App

A premium, fully responsive weather application with cinematic UI effects, real-time data, and smooth animations.

🔗 **Live Demo:** https://darshanchhetri24.github.io/weather-app/

---

## ✨ Features

* 🌍 **Auto Location Detection** (Geolocation API)
* 🔍 **City Search Functionality**
* 📊 **Hourly Forecast Slider**
* 📅 **7-Day Forecast**
* 🌙 **Dark / Light Mode Toggle**
* 🎬 **Cinematic Weather Effects**

  * Rain with motion
  * Thunder flashes ⚡
  * Sun glow ☀️
  * Moving clouds ☁️
  * Snow & fog effects
* 🎨 **Dynamic Sky Background (time-based)**
* 💎 **Glassmorphism Premium UI**
* 📱 **Fully Responsive (Mobile + Desktop)**

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Advanced UI + Animations)**
* **JavaScript (Vanilla)**
* Weather API from OpenWeatherMap

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/DarshanChhetri24/weather-app.git
```

2. Navigate to the project folder:

```bash
cd weather-app
```

3. Open `index.html` in your browser.

---

## 🔑 API Setup

1. Get your API key from OpenWeatherMap
2. Replace in `script`:

```javascript
const API_KEY = "YOUR_API_KEY";
```

---

## 🚀 Deployment

This project is deployed using GitHub Pages.

To deploy:

1. Push code to GitHub
2. Go to **Settings → Pages**
3. Select **main branch**
4. Save
